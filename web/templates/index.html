3-Layer Scanning Engine

1

Filename Scanner

Scans the .jar file names against a hardcoded list of known cheat client signatures (Wurst, Meteor, Sigma, Vape, Impact, Aristois, Wolfram, and more) using fast, case-insensitive string parsing.

2

Manifest & Package Check

Inspects the package directories and reads manifest files inside the archive (e.g. MANIFEST.MF, fabric.mod.json, mods.toml) to verify identifier packages and author tags.

3

Bytecode String Scan

Scans binary .class files and textual configuration files inside the mod for signature cheat strings (e.g. aimbot, killaura, freecam, scaffold, wallhack, fastplace, fly, bhop).
